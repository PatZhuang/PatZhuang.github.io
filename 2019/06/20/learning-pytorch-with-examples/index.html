Learning Pytorch With Examples

Posted on 2019-06-20

本文是 LEARNING PYTORCH WITH EXAMPLES 的学习笔记

本文将通过一些 self-contained 的示例介绍 PyTorch 的一些基本概念。

PyTorch 提供两个主要特性：

  • N 维的 Tensor, 和 numpy 的 ndarray 类似，但是可以运行在 GPU 上
  • 为神经网络的构建和训练提供自动求导

我们将使用一个全连接 ReLU 网络作为示例，该网络只有一个隐藏层，使用随机梯度下降法来训练。我们将最小化输出和真实值之间的欧几里得距离来拟合一些随机数据。

Tensors

Warm-up: numpy

在引入 PyTorch 之前，我们先使用 numpy 来实现这个网络。

Numpy 提供了一个 N 维数组对象和一些用于操纵这个数组的函数。Numpy 是一个用于科学计算的通用框架，它与计算图、深度学习和梯度没有直接关系。但我们可以很容易地使用 Numpy 编写前向和反向传播来实现一个拟合随机数据的两层网络。

PyTorch: Tensors

Numpy 很棒，但它无法使用 GPU 来加速运算。对于现代深度学习网络，GPU 通常要比 CPU 快 50 倍以上，因此很遗憾，Numpy 无法运用到现代深度学习上。

我们在此引入 PyTorch 最基础的概念：Tensor. 一个 PyTorch Tensor 和 Numpy Array 在概念是是完全相同的：Tensor 是一个 N 维数组，且 PyTorch 提供了对这个数组进行运算的多种函数。与 Numpy Array 不同的是，Tensor 可以追踪运算图和梯度，当然它也可以用于通用的科学计算。

Tensor 不同于 Numpy Array 的另一点在于，它可以使用 GPU 来加速数值计算。

Autograd

PyTorch: Tensors and autograd

在上面的示例中，我们需要手动实现神经网络的前向和反向传播。对于两层的小网络来说，手动实现反向传播问题不大，但是对于复杂的网络来说，手动实现反向传播就会变得很困难。

所幸我们可以使用自动微分（automatic differentiation）来自动完成反向传播阶段的梯度计算。PyTorch 的 autograd 包就提供了这样的功能。使用 autograd 后，在神经网络的前向传播的过程中会定义一张 计算图（computational graph），图的结点是 Tensor, 边是由输入 Tensor 生成输出 Tensor 的对应函数（function）. 对这张图执行反向传播可以很容易地计算梯度。

虽然听上去复杂，但是实现起来是很容易的。每个 Tensor 代表计算图中的一个结点。如果 x 是一个 Tensor 且它的 requires_grad 属性为 True, 则 x.grad 是指向 x 对于某个标量的梯度的 Tensor.

nn module

PyTorch: nn

计算图和自动梯度是定义复杂操作和自动计算微分的强力工具，但是对于大型的神经网络，原始的 autograd 就显得有点低级了。

构建神经网络的时候，我们通常会想把计算排列为层（layer），某些层有可学习参数（learnable parameters），这些参数可以在学习过程中被优化。

TensorFlow 的 Keras, TensorFlow-Slib 和 TFLearn 包中提供了相比原始的计算图更高层的接口来构建神经网络。在 PyTorch 中，这样的包是 nn.

nn 定义了一系列的 模块（Module），相当于神经网络中的层（Layer）。一个模块接受输入张量并计算输出张量，并保存中间状态例如有可学习参数的 Tensor. nn 也定义了一系列常用的损失函数。
