PyTorch Data Loading Tutorial

Posted on 2019-06-19

本文为 DATA LOADING AND PROCESSING TUTORIAL 的学习笔记

机器学习的预处理一般要花费很大的精力，PyTorch 为数据读取提供了很多方便的工具，也使得代码更加易读。本文将演示如何对一个数据集进行读取、预处理和数据增强。

本文需要使用的 package 有：

  • scikit-image: 数据读取和变换
  • pandas: 方便 csv 文件的 parsing

Dataset class

torch.utils.data.Dataset 是表示一个数据集的抽象类。自定义的数据集要继承 Dataset 类并重写以下方法：

  • __len__: 使 len(dataset) 返回数据集的大小
  • __getitem__: 使数据集支持下表访问如 dataset[i]

接下来为脸部标注数据集创建一个 dataset class. 我们将 csv 的读取放在 __init__ 中，把图片的读取放在 __getitem__ 中，这有助于减少内存占用。

我们将数据集的样本设置为一个字典：{'image': image, 'landmarks': landmarks}. 我们的数据集接受一个可选参数 transform.

Transforms

上述样本的一个显而易见的问题是，它们的尺寸是不一致的。大多数神经网络都要求输入数据有一致的尺寸，因此，我们需要对图片做一些预处理：

  • Rescale: 缩放图片
  • RandomCrop: 随机切割图片，这是数据增强的操作
  • ToTensor: 把 numpy image 转换为 torch image，这里要做轴变换

我们将使用 callable 类而不是普通的函数来实现这些功能，这样就无需在每次变换的时候都传入变换参数。我们要实现的方法有 __call__ 和 __init__ （如果需要），之后可以像这样使用这个变换：

tsfm = Transform(params)
transformed_sample = tsfm(sample)

注意这些变换要同时应用到图片和标注上。

以上的三个变换方法在 transform 库中都自带，分别叫 Resize, Rescale 和 ToTensor

Iterating through the dataset

结合以上的内容我们可以创造一个经过组合变换的数据集，这个数据集在每次被采样时：

  • 一张图片被读取
  • 对这张图片应用组合变换
  • 由于切割变换是随机的，因此在采样的同时也做了数据增强
0 torch.Size([3, 224, 224]) torch.Size([68, 2])
1 torch.Size([3, 224, 224]) torch.Size([68, 2])
2 torch.Size([3, 224, 224]) torch.Size([68, 2])
3 torch.Size([3, 224, 224]) torch.Size([68, 2])

为了解决简单 for loop 无法做到：

  • 将数据分批（Batching）
  • 数据洗混（shuffling）
  • 使用 multiprocessing 并行读取数据

可以使用 torch.utils.data.DataLoader 来读取数据。一个有意思的参数是 collate_fn, 可以用这个参数指定样本如何被分批（batch），但是使用默认参数在大多数情况下就足够了。
