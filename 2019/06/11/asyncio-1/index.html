启动一个 Coroutine

有多种方法可以使 asyncio 的事件循环（event loop）启动一个协程（coroutine），最简单的方式是调用 run_until_complete() 方法

即启动一个协程对象的方法是：获取一个 event loop，使用 run_until_complete 方法执行 coroutine 对象。

coroutine 对象在 return 的时候执行结束，所有 coroutine 对象执行结束后 run_until_complete 执行结束。

最后需要使用 close 方法关闭对应的 event loop.

需要注意的是，使用 get_event_loop 获取的事件循环对象是唯一的一个全局对象，在调用 close 方法后会被彻底关闭。如果在同一段代码中第二次调用 get_event_loop 方法会产生错误，需要重启 python 环境。可以使用 new_event_loop 方法来避免此问题。
